# ts2date
Convert Unix timestamps to human-readable dates. Written in Go.
```bash
curl -LO https://github.com/javimosch/ts2date/releases/latest/download/ts2date-linux-amd64
chmod +x ts2date-linux-amd64; sudo mv ts2date-linux-amd64 /usr/local/bin/ts2date
```
Usage: `ts2date 1700000000` → `2023-11-14T22:13:20Z`

---

This tool is a plugin for [supercli](https://github.com/javimosch/supercli) — an AI-friendly, config-driven dynamic CLI platform.
